# Challenge 5: Combining Slots, Properties, and Events

**Description:** Create a complex custom element that combines slots, properties, and events.

**Task:** Create a custom element `<challenge-5>` that displays a user profile with editable fields.

**Requirements:**

- Use Lit to define the custom element.
- The element should have name and email properties.
- The element should have slots for additional content (e.g., user bio).
- Provide an edit button that toggles between view and edit modes.
- Emit a custom event when the profile is updated.
