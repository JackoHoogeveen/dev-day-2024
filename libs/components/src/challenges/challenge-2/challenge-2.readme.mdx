# Challenge 2: Alert Web Component with Slots

**Description:** Create an alert web component that allows the default message to be overwritten with a slot.

**Task:** Create a custom element `<challenge-2>` that displays an alert message with the following features:

- **Slots:** A default slot for content and an optional title slot for a bold title above the content

**Requirements:**

1. Use Lit to define the custom element.
2. Use a default slot for the alert content, if no slotted content has been provided show the default message.
3. Use an optional title slot for a bold title above the content.

[Help](https://lit-element.readthedocs.io/en/v0.6.4/docs/templates/slots/)
